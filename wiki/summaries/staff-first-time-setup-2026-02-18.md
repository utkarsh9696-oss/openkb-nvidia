# staff-first-time-setup-2026-02-18.pdf

## Overview
The Unique Schools system is a comprehensive platform designed for school staff to access various features and tools. To access the system, staff members must be added to the school's database and complete the registration process through the Main School App. This guide provides a step-by-step explanation of the registration process, including how administrators add staff to the system, how staff download and register, and how to access all platforms.

## Key Concepts
* Unique Schools system
* Main School App
* Administrators App
* Desktop Dashboard
* Registration process
* Login with Microsoft option
* Additional access and permissions

## Main Topics Covered
### Initial Step: Ensure You Are Added to the System
This section explains the process for administrators to add staff to the system, including navigating to the Administration section, selecting Add/Edit Students & Staff, and entering staff details correctly.

### How Staff Register
This section outlines the registration process for staff, including downloading the Main School App, registering an account, and logging in across all platforms.

### Accessing All Platforms
This section explains how staff can access the Main School App, Administrators App, and Desktop Dashboard, including download links and login instructions.

### Login with Microsoft
This section provides alternative registration steps for staff who use the "Login with Microsoft" option, including downloading the Main School App and logging in with a school Microsoft account.

## Important Relationships and Connections
The Unique Schools system is comprised of three platforms: the Main School App, Administrators App, and Desktop Dashboard. Staff must complete the registration process through the Main School App before accessing the other platforms. The "Login with Microsoft" option provides an alternative registration method for staff who use Microsoft accounts. Additional access and permissions can be requested by emailing staffsupport@uniqueschools.ie.

## Key Takeaways
* Staff must be added to the school's database before registering for the Unique Schools system.
* The registration process must be completed through the Main School App.
* Staff can access the Main School App, Administrators App, and Desktop Dashboard once registered.
* Additional access and permissions can be requested by emailing staffsupport@uniqueschools.ie.
* The "Login with Microsoft" option provides an alternative registration method for staff who use Microsoft accounts.
